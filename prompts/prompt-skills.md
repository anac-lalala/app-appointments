# Skills Roadmap Prompt

Actúa como mentor técnico senior y ayúdame a definir qué habilidades necesito dominar para construir correctamente este MVP de gestión de citas mientras aprendo buenas prácticas reales.

Quiero una guía:
- práctica
- enfocada en este proyecto
- priorizada
- sin teoría excesiva
- útil para aprender mientras implemento

---

## Contexto del proyecto

Voy a construir un MVP con:
- Next.js App Router
- Tailwind
- FastAPI
- SQLAlchemy async
- PostgreSQL
- Alembic
- OTP por email
- JWT
- Docker

Quiero aprender:
- modularidad
- SOLID práctico
- buenas prácticas reales
- arquitectura comprensible
- testing razonable

---

## Lo que quiero que produzcas

### 1. Mapa de habilidades
Qué habilidades necesito para construir este proyecto bien.

### 2. Priorización
Qué debo aprender primero, segundo y después.

### 3. Just enough knowledge
Qué debo aprender solo hasta el nivel necesario para este MVP.

### 4. Habilidades backend
Qué conceptos backend son clave.

### 5. Habilidades frontend
Qué conceptos frontend son clave.

### 6. Habilidades transversales
Qué conceptos de arquitectura, testing, seguridad y Docker necesito.

### 7. Qué no estudiar todavía
Qué temas sería mejor posponer.

### 8. Ruta de aprendizaje ligada al proyecto
Cómo aprender mientras implemento funcionalidades reales.

### 9. Errores de aprendizaje comunes
Qué evitar para no atascarme estudiando demasiado.

### 10. Propuesta final
Cierra con una ruta concreta de aprendizaje para este MVP.

---

## Restricciones

- no roadmap académico enorme
- no listas infinitas de temas
- no recomendar aprender 20 cosas antes de empezar
- sí enfoque just-in-time

---

## Resultado esperado

La respuesta debe terminar con:
1. top habilidades prioritarias
2. qué aprender antes de codificar
3. qué aprender mientras construyo
4. qué dejar para después del MVP
